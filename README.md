# Noir Bistro

Immersive restaurant demo workspace for premium hospitality clients.

## What this is

- A luxury restaurant landing page concept
- A reusable client demo foundation
- A local-first static build that does not need a framework install

## Files

- `index.html` - page structure
- `styles.css` - visual system and layout
- `src/main.js` - background animation and reveal logic
